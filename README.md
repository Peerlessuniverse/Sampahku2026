# SampahKu 2026: Cosmic Waste Management Ecosystem

**SampahKu 2026** adalah platform pengelolaan limbah cerdas dengan estetika **Premium Cosmic & Forest**, dirancang untuk mendigitalisasi ekosistem pengelolaan sampah melalui pendekatan yang inovatif dan visual yang memukau.

## ğŸš€ Fitur Utama

### 1. Management Portal (Control Portal)
Pusat kendali admin dan partner untuk mengelola ekosistem aplikasi secara real-time.
- **Data Table Terpadu:** Manajemen partner/sponsor dalam format tabel yang ringkas dan profesional.
- **Smart Filtering:** Filter data berdasarkan Nama Brand, Level Paket (Nebula, Galactic, Cosmic), Status Kontrak, dan Tanggal Pendaftaran.
- **Performa Real-time:** Pantau jumlah impresi (view) dan interaksi sponsor secara langsung.

### 2. Waste-to-Energy (WTE) Simulator
Modul edukatif yang mensimulasikan transformasi limbah menjadi sumber energi terbarukan.
- **Siklus Termal Plastik.**
- **Simbiosis Baterai Organik.**
- **Microbial Fuel Cell (MFC).**

### 3. SampahKu Scanner
Teknologi pemindaian cerdas untuk identifikasi jenis limbah dan panduan pemilahan yang tepat.

### 4. Carbon Footprint Calculator
Alat ukur jejak karbon personal untuk meningkatkan kesadaran dampak lingkungan.
